🛒 E-Commerce Website using Flask
A simple and professional E-commerce web application developed using Python Flask, SQLite, HTML, and CSS.
The project includes user authentication, product browsing, cart functionality, and an admin panel for managing products.
📌 Features
👤 User Features
User registration and login
View all products
Search products
Add products to cart
Remove products from cart
View total cart price
Secure session-based authentication
🛠 Admin Features
Admin login (separate from users)
Admin dashboard
Add new products
Delete existing products
View all products
🧑‍💻 Technologies Used
Backend: Python (Flask)
Frontend: HTML, CSS
Database: SQLite
Icons: Font Awesome
Styling: Custom CSS with animations
