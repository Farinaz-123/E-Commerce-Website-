E-COMMERCE WEBSITE USING FLASK

This is a simple E-commerce web application developed using Python Flask and SQLite.
The project allows users to register, login, view products, add items to a cart, and
provides an admin panel for managing products.

--------------------------------------------------

FEATURES

User Features:
- User registration
- User login
- View products
- Search products
- Add products to cart
- Remove products from cart
- View cart total
- Logout

Admin Features:
- Admin login
- Admin dashboard
- Add products
- Delete products

INSTALLATION STEPS

1. Clone the repository or download the project.
2. Open the project folder in VS Code.
3. Install dependencies using:
   pip install -r requirements.txt
4. Run the application using:
   python app.py
5. Open browser and visit:
   http://127.0.0.1:5000/login
